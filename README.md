# Smart Lab Management System 🔬
This is my **BCA 6th Semester Project**—a robust management system built using Java and the Spring Boot ecosystem to streamline, track, and automate laboratory operations.

---

## 📋 Project Introduction & Database Setup
All detailed documentation, features, system architecture diagrams, and the **database SQL dump file** are centrally managed inside the `docs` folder.

👉 **[View Project Documentation & SQL Schema](./docs/)**

---

## 🛠️ Tech Stack
* **Language:** Java
* **Framework:** Spring Boot (MVC / REST API)
* **Build Tool:** Maven (`pom.xml`)
* **Database:** SQL-based relational database

---

## 📂 Repository Structure
* `/docs` — Contains the project introduction, system synopsis, and the `.sql` script to set up your database tables.
* `/src/main` — Core Java source code, business logic packages, and application configurations.
* `pom.xml` — Project Object Model containing all Maven dependencies and plugins.

---

## ⚙️ Quick Start
1. Clone this repository to your local machine.
2. Navigate to the `/docs` folder, find the `.sql` file, and import it into your local database server.
3. Open the project in your IDE (IntelliJ IDEA recommended).
4. Update the database credentials in your application configuration files under `/src/main/resources`.
5. Run the Spring Boot application.
